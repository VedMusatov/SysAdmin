# SysAdmin

За полода проведен Полный цикл модернизации IT-инфраструктуры предприятия: от аудита хаотичной сети «домашнего уровня» до проектирования и внедрения отказоустойчивой корпоративной сети с профессиональным оборудованием MikroTik.
Работа выполнена наживую, те исключая простои.
Дополнительно все задокументировано и обучен вновь принятый сотрудник нулевой джун до уровня четверть-мидл, для последующей поддержки.

Ключевые достижения
1. Кардинальная перестройка физической топологии сети
Ликвидация «бутылочных горлышек»: Демонтаж устаревшей инфраструктуры, основанной на последовательном соединении неуправляемых хабов и роутера домашнего класса.
Внедрение иерархической модели: Переход на звездообразную топологию с четким разделением на ядро (Core), уровень распределения (Distribution) и уровень доступа (Access).
Апгрейд каналов связи: Организация высокоскоростных магистралей между коммутаторами и ядром с использованием оптических интерфейсов SFP+ (1-10G), что обеспечило пропускную способность, достаточную для любых нагрузок, в частности виртуализации и видеонаблюдения.

2. Проектирование логической структуры и VLAN-архитектуры
Сегментация трафика (VLANs): Разделение единого широковещательного домена на изолированные логические сегменты для повышения безопасности и производительности:
-Выделенный сегмент для критичной инфраструктуры (AD, SQL, Mail).
-Изолированная подсеть для IP-телефонии Asterisk, обеспечивающая приоритизацию голосового трафика (QoS).
-Отдельные сегменты для склада и систем безопасности (ACS, видеонаблюдение).
-Изолированная WiFi сеть для мобильных устройств сотрудников и гостей.

Маршрутизация: Настройка брандмауэра, межсетевого экранирования и правил маршрутизации на шлюзе MikroTik CCR 2004-16, ограничение доступа между сегментами (например, запрет доступа из гостевого WiFi в рабочий сегмент).

3. Внедрение серверной инфраструктуры и виртуализации
Централизация управления: Развертывание контроллера домена Active Directory (AD DC) и DNS-сервера для централизованного управления учетными записями и политиками безопасности (GPO на базовом уровне через группы).
Виртуализация (VDS): Миграция некоторых физических серверов на платформу виртуализации (Hyper-V), выделение ресурсов под тестовые среды и продуктивные сервисы.
Отказоустойчивость: Настройка резервного контроллера домена, SQL, файлового и терминальных серверов для обеспечения непрерывности бизнес-процессов.

4. Интеграция специализированных систем
IP-Телефония: Полный переход с аналоговой связи на цифровую АТС Asterisk, настройка SIP-транков и DECT-трубок.
Системы безопасности (ACS): Интеграция контроллеров доступа (СКУД) и домофонии в единую сеть с выделенным сервером ACS.
Видеонаблюдение: Выделение отдельной подсети для IP-камер, настройка потоковой передачи видео без нагрузки на пользовательский трафик.

5. Настройка почтового сервера на MS Exchange вместо почты на Mail.ru, включая получение сертификата, mx и ptr записи.

6. Переделана система бэкапов для максимальной защищенности баз и файлов.

7. Настроены сервер мониторинга сети и частично уведомления от серверов через скрипты PS.

8. Все это происходило во время работы, изначально в одиночку, потом с помощником. Мелочи типа настроек ТСД, ЭЦП, перенос локальных принтеров на принтсервер и тд

Технический стек
Сетевое оборудование: MikroTik (CCR 2004-16, CRS328-24P-4S+RM, CRS112-8P-4S-IN), D-Link DGS-1250-52.
Серверные ОС и сервисы: Windows Server 2008-2026 (AD DS, DNS, DHCP, RDS), Linux (Asterisk, Mail Server), Virtualization (VDS/Hypervisor).
Протоколы и технологии: OSPF/Static Routing, VLAN (802.1Q), SFP+, L2TP/IPsec VPN, SIP/DECT.

Результат работы
Сеть переведена из состояния «нестабильная и небезопасная» в состояние «Enterprise-ready».
Исчезли проблемы с коллизиями и широковещательными штормами.
Обеспечена возможность масштабирования (добавление новых этажей/складов без перестройки ядра).
Повышен уровень безопасности за счет изоляции критичных данных и систем видеонаблюдения от пользовательских сегментов.
